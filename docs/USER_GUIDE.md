# Руководство пользователя

## Введение

Система управления полисами предназначена для автоматизации работы страхового брокера. Она позволяет вести учет полисов, управлять графиками платежей и получать уведомления о предстоящих оплатах.

## Основные разделы

### 1. Дашборд

Главная страница системы отображает:
- Количество активных полисов
- Предстоящие платежи (в течение 30 дней)
- Просроченные платежи
- Недавно добавленные полисы

### 2. Журнал полисов

Полный список всех полисов с возможностью фильтрации по:
- Номеру полиса
- Номеру ДФА
- Клиенту
- Страховщику
- Филиалу
- Виду страхования
- Статусу

**Действия:**
- Просмотр детальной информации
- Редактирование
- Экспорт в Excel

### 3. Карточка полиса

Детальная информация о полисе:
- Основные данные (номера, клиент, страховщик)
- Финансовая информация (стоимость имущества, премия, франшиза)
- График платежей
- Статусы (полис, ДФА)

### 4. График платежей

Список всех платежей с фильтрацией:
- Все платежи
- Предстоящие (в течение 30 дней)
- Просроченные
- Оплаченные

Для каждого платежа отображается:
- Полис и клиент
- Дата платежа
- Сумма
- Комиссия (КВ)
- Статус оплаты

### 5. Клиенты

Справочник клиентов с возможностью:
- Просмотра списка
- Поиска по названию или ИНН
- Просмотра всех полисов клиента

### 6. Отчеты

Экспорт данных в Excel:
- Все полисы
- График платежей

## Работа с полисами

### Создание нового полиса

1. Перейдите в Админ-панель → Полисы → Добавить полис
2. Заполните обязательные поля:
   - Номер полиса
   - Клиент (лизингополучатель)
   - Страховщик
   - Описание имущества
   - Даты начала и окончания
   - Стоимость имущества
   - Вид страхования
   - Филиал
3. Добавьте график платежей (inline)
4. Сохраните

### Редактирование полиса

1. Найдите полис в журнале
2. Нажмите кнопку "Редактировать"
3. Внесите изменения
4. Сохраните

### Добавление платежей

Платежи добавляются в карточке полиса:
1. Откройте полис на редактирование
2. В разделе "График платежей" добавьте строки
3. Укажите:
   - Год и номер платежа
   - Дату платежа
   - Сумму
   - Ставку комиссии (выбирается из справочника)
4. Комиссия в рублях рассчитается автоматически
5. Сохраните

### Отметка об оплате

1. Найдите платеж в графике
2. Откройте на редактирование
3. Укажите фактическую дату оплаты
4. При необходимости укажите дату согласования СК
5. Сохраните

## Справочники

### Страховые компании

Список страховщиков, с которыми работает брокер.

### Филиалы

Филиалы лизинговой компании для распределения полисов.

### Виды страхования

Типы страхования: КАСКО, Имущество, Спецтехника и др.

### Ставки комиссий

Процент комиссии брокера для каждой комбинации:
- Страховщик
- Вид страхования

Используется для автоматического расчета комиссии в платежах.

### Инфо-метки

Метки для гибкой классификации полисов (Инфо 1 и Инфо 2).

## Уведомления

Система автоматически проверяет:
- Предстоящие платежи (за 7, 3 и 1 день)
- Просроченные платежи

Email-уведомления отправляются менеджерам (настраивается в админ-панели).

## Экспорт данных

### Экспорт полисов

1. Перейдите в раздел "Отчеты"
2. Нажмите "Скачать Excel" в блоке "Экспорт полисов"
3. Файл содержит все полисы со всей информацией

### Экспорт платежей

1. Перейдите в раздел "Отчеты"
2. Нажмите "Скачать Excel" в блоке "Экспорт графика платежей"
3. Файл содержит все платежи по всем полисам

## Советы по работе

1. **Используйте фильтры** в журнале полисов для быстрого поиска
2. **Проверяйте дашборд** ежедневно для контроля предстоящих платежей
3. **Настройте ставки комиссий** перед созданием полисов для автоматического расчета
4. **Используйте инфо-метки** для дополнительной классификации полисов
5. **Регулярно экспортируйте данные** для резервного копирования

## Часто задаваемые вопросы

**Q: Как изменить статус полиса?**
A: Откройте полис на редактирование и измените флажки "Полис активен" или "ДФА активен".

**Q: Почему не рассчитывается комиссия?**
A: Убедитесь, что в справочнике "Ставки комиссий" есть запись для комбинации страховщика и вида страхования.

**Q: Как найти все полисы клиента?**
A: Откройте карточку клиента - там отображаются все его полисы.

**Q: Можно ли удалить полис?**
A: Да, через админ-панель, но рекомендуется просто закрывать полисы (снимать флажок "Активен").

**Q: Как настроить уведомления?**
A: Настройка производится через Celery Beat в админ-панели (требуется установка Redis).
